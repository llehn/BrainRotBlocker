**BrainRotDoctor** is a free, open-source Windows app that curbs Instagram —
and TikTok, YouTube Shorts, Reddit, X, Facebook Reels, … — doom-scrolling while
keeping the useful parts reachable (Direct Messages, Stories). You define rules
for *what* to limit and *when*: a per-hour time allowance or a full block, on the
days and within the hours you choose. It watches your browser tabs through
Windows UI Automation, accounts your time against the rules, and closes a tab
when an active rule says you're out of time. An optional **strict mode** locks
your settings for a duration you pick, behind a double opt-in, so you can't
casually undo it in a weak moment.

The UI is localized into **31 languages** and follows your Windows display
language automatically.

### Install — Windows 10/11, no admin needed

1. Download **`BrainRotDoctor.exe`** below and double-click it.
2. Windows SmartScreen will warn about an unknown publisher (the app is unsigned
   and open source). Click **More info → Run anyway**.
3. Click **Install** in the prompt.

It installs under your user folder, starts with Windows, and adds an *Apps &
features* entry for clean uninstallation. More details and screenshots:
https://llehn.github.io/BrainRotDoctor/
